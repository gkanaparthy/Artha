# Artha Branding Assets

This directory contains the primary brand assets for Artha.

## Assets

- `artha-logo-primary.png`: The primary organic leaf/seed mark in Forest Green (#2E4A3B). Used in headers, sidebars, and login screens.
- `artha-social-preview.png`: The 4K Open Graph image optimized for social media sharing (LinkedIn, X/Twitter, etc.).

## Brand Colors

- **Primary Forest Green**: `#2E4A3B`
- **Soft Cream Background**: `#FAFBF6`
- **Accent Peach**: `#E59889`
- **Sage Support**: `#E8EFE0`

## Typography

- **Headings**: Playfair Display (Serif)
- **Body**: Inter (Sans-serif)
